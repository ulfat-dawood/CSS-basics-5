Types of units

  • Percentage

Percentage

  • Percentage based on parent element.
  • Mostly used to set responsive width.

Absolute Units

  • cm - centimeters
  • mm - millimeters
  • in - inches
  • pt - points
  • px - pixels

Pixels (px) are used mostly, for all measures.
Although px are fixed and will appear the same for different viewport, they are relative to the viewing device.

Height, width and font-size of this box will remain the same accros all window size.
